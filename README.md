# Crypto Exchange static site

Полностью статический сайт без npm, React и сборки.

## Netlify настройки для GitHub

- Build command: оставить пустым
- Publish directory: `.`

## Ручной деплой

Перетащить папку с `index.html` в Netlify Deploy manually.

## Курсы

Курсы статические внутри `index.html` в объекте `ratesUsd`.
